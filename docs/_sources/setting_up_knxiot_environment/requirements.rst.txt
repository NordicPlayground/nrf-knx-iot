.. _requirements:

Requirements
############

This page outlines the requirements that you need to meet before you start working with the KNX IoT Point API environment.

Software requirements
*********************

To meet the software requirements, install the following tools:

* `nRF Connect SDK v2.3.0`_
* `nRF Command Line Tools`_
* `Wireshark`_ (for tracing/sniffing purposes)

Hardware requirements
*********************

To meet the hardware requirements, ensure you have the following:

* The Nordic's `nRF52840 Development Kit`_ needed to run :ref:`light_switch_actuator` and :ref:`light_switch_sensor` samples
* The Nordic's `nRF52840 Dongle`_ as the Thread sniffer
